# Asistente Académico Inteligente

Aplicación web para gestionar tareas académicas utilizando inteligencia artificial y automatización.

## Objetivo del Proyecto

Desarrollar una aplicación que permita a estudiantes organizar tareas académicas, consultar información y mejorar su productividad mediante herramientas inteligentes.

## Tecnologías Utilizadas

- Python
- FastAPI
- HTML
- JavaScript
- Git y GitHub

## Estructura del Proyecto

/backend → lógica del servidor y API  
/frontend → interfaz de usuario  
README.md → documentación del proyecto  
requirements.txt → dependencias necesarias
