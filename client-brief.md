# Client Brief

## Problema
Se requiere desarrollar una aplicación como parte de un proyecto académico que se construirá de forma progresiva durante el semestre.

## Objetivo
Crear una solución funcional aplicando buenas prácticas de desarrollo, control de versiones y uso de herramientas como Claude Code.

## Alcance inicial
- Configuración del entorno
- Estructura del proyecto
- Documentación inicial
- Primer avance de código

## Metodología
- Uso de GitHub Issues para organizar tareas
- Desarrollo incremental
- Uso de Claude Code como asistente

## Estado del proyecto
Fase inicial (entrega parcial del semestre)
